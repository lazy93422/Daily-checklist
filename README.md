# 每日习惯打卡

一个简单、轻量的每日习惯打卡网页应用。它可以记录每天的打卡情况，支持自定义打卡项目、查看统计数据、按日历查看历史记录，并可导出 Excel 数据。

在线访问地址：

https://lazy93422.github.io/Daily-checklist/

## 主要功能

- 每日打卡：勾选或取消勾选当天的习惯项目。
- 自定义项目：可以新增、编辑、删除自己的打卡内容。
- 统计页面：查看累计打卡天数、累计勾选次数，以及每个项目被打卡的天数。
- 趋势图：以日期为横轴、当天勾选数量为纵轴，展示打卡变化趋势。
- 日历详情：在统计页面点击具体日期，查看当天的详细打卡情况。
- 数据导出：可将历史打卡数据导出为 Excel 可打开的 `.xls` 文件。
- 手机安装：支持作为 PWA 添加到手机主屏幕，像简单 App 一样使用。
- 离线缓存：安装或访问后，基础页面可在离线状态下打开。

## 默认打卡项目

应用初始包含以下四个项目：

- 今天没抽烟
- 今天没喝酒
- 今天运动了
- 今天早睡了

这些项目可以在页面中的“管理打卡内容”区域修改。该区域默认折叠，点击后可以展开。

## 使用方式

1. 打开网页。
2. 在“今日打卡”页面勾选当天完成的项目。
3. 如需修改打卡内容，展开“管理打卡内容”。
4. 点击“统计”选项卡查看累计数据、趋势图和日历详情。
5. 如需备份数据，点击“导出 Excel”。

## 手机安装

### Android

1. 使用 Chrome 打开在线访问地址。
2. 点击浏览器菜单。
3. 选择“添加到主屏幕”或“安装应用”。
4. 按提示完成安装。

### iPhone

1. 使用 Safari 打开在线访问地址。
2. 点击底部分享按钮。
3. 选择“添加到主屏幕”。
4. 点击“添加”。

## 数据说明

打卡数据保存在当前设备浏览器的 `localStorage` 中。

这意味着：

- 数据不会上传到服务器。
- 不同手机、电脑或浏览器之间不会自动同步。
- 清除浏览器数据可能会删除打卡记录。
- 导出的 Excel 文件可作为本地备份。

## 部署方式

本项目是纯静态网页，可以直接部署到 GitHub Pages、Netlify、Vercel 或其他静态网站托管平台。

部署到 GitHub Pages 时，需要上传以下文件：

```text
index.html
manifest.webmanifest
sw.js
assets/app-icon.svg
README.md
```

上传后，在 GitHub 仓库的 `Settings > Pages` 中开启 GitHub Pages，即可获得公开访问链接。

如果网页更新后手机端仍显示旧版本，可以在网址后添加版本参数，例如：

```text
https://lazy93422.github.io/Daily-checklist/?v=5
```

## 技术说明

- `index.html`：应用主页面，包含界面、交互逻辑、统计图和数据导出功能。
- `manifest.webmanifest`：PWA 配置文件，用于支持手机安装。
- `sw.js`：Service Worker 文件，用于缓存页面资源。
- `assets/app-icon.svg`：应用图标。

## 适用场景

这个应用适合个人记录日常习惯，例如戒烟、戒酒、运动、早睡、喝水、阅读等。它没有账号系统，也不依赖后端服务，更适合个人轻量使用。
